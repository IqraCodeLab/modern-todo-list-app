# ✨ Modern To-Do List App

A sleek, responsive, and feature-rich To-Do List application built with HTML, CSS (Vanilla), and JavaScript. Featuring a modern **Glassmorphism** design and optimized for a premium user experience.



## 🚀 Features

- **💎 Glassmorphism UI**: A stunning, semi-transparent design with backdrop-blur effects.
- **📊 Real-time Counter**: Keep track of your total and completed tasks at a glance.
- **🧹 Clear All**: Quickly wipe your entire list with a confirmation prompt.
- **💾 Persistent Storage**: Your tasks are saved automatically to your browser's local storage.
- **⌨️ Keyboard Support**: Press "Enter" to add tasks instantly.
- **📱 Responsive Design**: Works beautifully on desktops, tablets, and phones.
- **✨ Smooth Animations**: Subtle fade-in and hover transitions for a premium feel.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Modern styles, glassmorphism, and animations.
- **JavaScript (ES6+)**: Dynamic task management and local storage integration.
- **Google Fonts**: "Inter" for a clean, professional typography.

## ⚙️ How to Use

1. **Add a Task**: Type your task in the input field and click "Add Task" or press "Enter".
2. **Toggle Completion**: Click on any task to mark it as completed (it will be crossed out).
3. **Delete a Task**: Click the "×" icon on the right side of a task to remove it.
4. **Clear All**: Click the "Clear All" button below the input to reset your list.

## 📥 Installation

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
